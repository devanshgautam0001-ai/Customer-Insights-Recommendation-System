# Customer Insights & Recommendation System

## Overview

This project analyzes customer purchasing behavior using the Olist E-Commerce Dataset.

Features:

- Customer Segmentation using KMeans
- Revenue Analysis
- Payment Analysis
- Product Category Analysis
- Customer State Analysis
- Business Recommendations
- Interactive Dashboard using Streamlit

## Tech Stack

- Python
- Pandas
- Plotly
- Streamlit
- Scikit Learn

## Machine Learning

KMeans Clustering

Segments:

- Premium
- Loyal
- Regular
- At Risk

## Dataset

Brazilian Olist Ecommerce Dataset

## Run

pip install -r requirements.txt

streamlit run app.py

# 📌 Project Overview

Customer Insights & Recommendation System is an interactive Business Intelligence Dashboard that analyzes customer purchasing behavior, payment methods, product categories, customer segments, revenue, and order trends.

The project uses the Brazilian Olist E-Commerce Dataset and applies data cleaning, feature engineering, customer segmentation, and visualization techniques to generate business insights and actionable recommendations.

This project was developed as part of the **ReadyNest Internship Program**.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/devanshgautam0001-ai/Customer-Insights-Recommendation-System.git
```

Go inside project

```bash
cd Customer-Insights-Recommendation-System
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
